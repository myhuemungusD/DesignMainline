Workflow:
  • Relief: portraits, logos with depth — print flat on bed, layer height 0.12–0.16mm.
  • Lithophane: photos lit from behind — print vertical with brim, white PLA, 0.1mm layers, 100% infill, ~3mm thick.
  • Logo emboss: small relief (1–2mm) on a thick base — good for nameplates, badges.
  • Cut to subject: turn on Background Removal first, then enable "Cut to subject" — mesh follows the silhouette, no rectangular base. Best with soft edges OFF for crisp outlines.
  • Blender: import STL or OBJ → Edit Mode → smooth/decimate as needed → export new STL.
  • Cura: File → Open → STL → orient flat or vertical → slice → save G-code.
Resolution tradeoff: 200 is a good default. 300+ produces big files (10+ MB) and slow Cura slices. 100 is fine for simple logos.
