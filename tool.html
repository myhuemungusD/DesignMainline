Why I Built This

I 3D-print a lot. The bottleneck was always the same — turning a flat image into a usable mesh.

Existing tools were either expensive subscription SaaS, broken open-source experiments, or marketplaces that wanted a cut of every download. I wanted something fast, no-account, no-subscription — drop an image, see it in 3D, pay once, get the file. So I built it.

iGrinder gives you a clean heightmap mesh with proper walls and a flat base — manifold, watertight, ready for the next step. Edit it in Blender (decimate, smooth, boolean cut, add text), then slice in Cura and print. That's the loop.

— Jason Hamilton · Design Mainline

The 5-Step Workflow
  1. Upload — PNG or JPG. Higher contrast = better depth. Flat lighting helps.
  2. Tune — pick a preset, then dial width, base, and relief height in mm. Live 3D preview updates instantly.
  3. Pay & Download STL — $9 unlocks both STL and OBJ. Yours to keep.
  4. Edit in Blender (optional) — Import STL → Edit Mode → smooth, decimate, boolean-cut, add base text, then export a new STL.
  5. Slice in Cura — File → Open → STL → orient flat (relief/emboss) or vertical (lithophane) → slice → save G-code → print.
Preset Cheat Sheet
  • Relief: portraits & logos with depth — print flat, 0.12–0.16 mm layers, any color PLA.
  • Lithophane: photos lit from behind — print vertical with brim, white PLA, 0.1 mm layers, 100% infill, ~3 mm thick.
  • Emboss: small relief (1–2 mm) on a thick base — nameplates, badges, signage.
  • Cut to Subject: turn on Background Removal first, then enable to follow the silhouette — no rectangular base.
Resolution above 300 produces files over 10 MB and slows slicing. 200 is the recommended default.
